Cách chạy ứng dụng:
- Cài đặt NodeJS
- Cài thêm gói http-server: tại terminal gõ lệnh "npm install -g http-server"
- Di chuyển đường dẫn đến thư mục chứa file index.html
- Tại terminal gõ "http-server"
- Sử dụng đường dẫn: 127.0.0.1:8080 để chạy và kiểm tra ứng dụng
- Kết thúc việc chạy ứng dụng: tại terminal, nhấn tổ hợp phím Ctrl+C
