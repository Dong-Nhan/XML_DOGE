**Hướng dẫn sử dụng:**

- Cài đặt nodejs, npm

- Cài đặt http-server:
```
npm install http-server -g
```
(hoặc `sudo npm install http-server -g`)

- Ở thư mục chứa file `index.html` chạy lệnh:
```
http-server -p <port>
```
(port là số hiệu port, mặc định là 8080)

- Mở trình duyệt web, gõ url:
```
localhost:<port>
```
(Ví dụ: `localhost:8080`)
